National Park Dashboard

This site allows the user to select a Colorado national park. They can see some base info of the park along with some images. The user can also see the weather forecast for the park of their choice.


I worked alongside four other developers to make this, where I focused on the JS and some CSS for the weather page.

Password Generator

This is a password generator where the user can select the constraints for the password (such as length, what kind of characters they want, and so on) then using those constraints the site will generate a random password and put it into a text space for the user to copy.

Weather Dashboard

A site that allows the user to search for a city and get the weather info of that city. It will show the current weather along with the forecast for the next 5 days. The searched cities will be saved allowing the user to get the info for that city without having to search for it again.
